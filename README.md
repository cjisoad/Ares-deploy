# Ares_deploy

这是一个围绕 Ares 机器人展开的工作区，当前有效内容主要分成三块：Ares 的 MuJoCo 仿真、Ares 的原始模型资料，以及历史归档参考。

## 目录结构

- `Ares_mujoco_deploy/`：当前主要功能仓库，提供 Ares 的 MuJoCo Python 仿真、模型转换脚本和控制示例。这里是实际运行入口，不是历史参考区。
- `model/`：Ares 的模型源数据目录，包含 URDF、网格、配置和启动文件等。它更像是仿真生成和模型维护的输入，不是独立应用入口。
- `archive/`：归档参考目录，只保留旧项目和第三方依赖的历史文件，用于查阅和对照，不参与当前功能运行或构建。

## 说明

- 根目录 README 只做总览，不重复子目录里的具体使用步骤。
- 当前开发和运行重点在 `Ares_mujoco_deploy/`，`archive/` 仅作历史资料保留。
