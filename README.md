# Ares_deploy

这是 Ares 机器狗的仿真与模型工作区。当前有效内容以 Ares 机型为中心，主要服务于 MuJoCo 仿真、模型维护和控制实验。

根目录只做总体索引；具体运行命令、脚本入口和控制细节见各功能目录内的 README 或源码。

## 当前结构

- `Ares_mujoco_deploy/`

  当前主要功能目录，包含 Ares 的 MuJoCo 仿真环境、运行时 MJCF、Python 仿真接口、控制模块和示例脚本。

  其中 `sim/` 负责 MuJoCo 加载、步进、状态读取和 MIT/PD 力矩接口；`control/` 放置状态机位置控制、步态相位、支撑腿/摆动腿控制、Pinocchio IK 和遥操作输入；`examples/` 提供自由下落、挂起踏步、状态机和 MIT 控制等验证入口；`tools/` 用于从模型源数据生成仿真资源；`assets/` 保存仿真直接加载的 MJCF 文件。

- `model/`

  Ares 的模型源数据目录，保存 URDF、mesh、配置和模型说明。该目录是仿真模型转换和校准的输入来源，不是主要运行入口。

- `archive/`

  历史归档和参考资料目录。这里的内容只用于查阅旧实现、第三方依赖或迁移参考，不参与当前 Ares 功能运行、导入或构建。

## 开发边界

- 当前运行和调试优先在 `Ares_mujoco_deploy/` 内进行。
- 模型几何、关节限位和姿态角度等源数据优先维护在 `model/`。
- `archive/` 不作为功能代码修改目标，除非明确是在整理历史资料。
